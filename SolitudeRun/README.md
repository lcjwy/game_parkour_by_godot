# SolitudeRun

Godot 4.6.3 stable / GDScript / Web-first 单机第三视角小汽车跑酷原型。

## 当前实现

- `scenes/`：主菜单与游戏主场景。
- `scripts/`：UI、车辆、道路、天气、资源类型、对象池。
- `resources/`：2 张地图、3 个程序化音效预设、买断制价格策略。
- `autoload/`：设置、多语言、音频、存档、全局状态、Steam 服务占位。

玩法规则：

- 可选择 `沙漠` 和 `雨林` 两张地图。
- 可选择 3 种默认音效：荒漠风声、雨林雨声、低沉引擎。
- 小汽车第三视角，单行弯曲车道。
- 单局需要坚持 `10800` 秒，也就是 3 小时。
- 游戏中只显示已游玩时间，不显示总目标时长。
- 不能暂停；`ui_cancel/Esc` 会被忽略。
- 若窗口失焦或连续 `8` 秒无操作，判定跑酷失败。
- 1 小时时提示：`已经坚持1小时`。
- 2.5 小时时提示：`放弃也许会更好，欢迎再次体验`。
- 失败弹窗：`未达到终点，放弃也许是一种更好的选择。`
- 成功弹窗：`你也许浪费了3小时，但是超越孤独的勇者，人生的意义就在对抗孤独。`
- 多语言：中文、英文、韩文。
- 买断制价格策略：`CN/CHN = 1 RMB`，其他地区 `1 USD`。

## 运行方式

你当前未安装 Godot 环境，所以本项目先以源码方式生成。

安装 Godot 4.6.3 stable 后：

1. 打开 Godot。
2. Import `SolitudeRun/project.godot`。
3. 运行主场景 `res://scenes/ui/main_menu.tscn`。
4. Web 导出使用 `export_presets.cfg` 里的 `Web` preset。

## 技术约束

- 不使用 C#/.NET，优先保证 Web 目标兼容性。
- 道路使用 `ArrayMesh` 生成可见窗口。
- 道路边缘标记使用 `MultiMesh` 批量绘制。
- 雨林天气和沙漠孤烟使用少量 `GPUParticles3D`，不创建海量节点。
- `ObjectPool` 已预留给后续障碍物、粒子提示或可复用装饰。
